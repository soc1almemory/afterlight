# Руководство пользователя

Afterlight помогает быстро фиксировать задачи, раскладывать их по дням и категориям, хранить короткие заметки и управлять задачами из Telegram.

## Основные разделы

- **Входящие** — место для задач без конкретного дня. Сюда удобно складывать всё, что нужно разобрать позже.
- **Сегодня** — задачи текущего дня. Если включена настройка просроченных задач, здесь также показываются активные задачи с прошедшей датой.
- **Неделя** — планирование по дням. Задачи можно добавлять сразу в конкретный день или перетаскивать между днями.
- **Категории** — контексты вроде “Учёба”, “Дом”, “Работа”, “Проект”. Категории можно добавлять в избранное.

## Работа с задачами

- Нажмите “Добавить задачу”, чтобы создать задачу в текущем разделе.
- У задачи можно указать название, описание, дату, время, приоритет и категорию.
- Нажмите на название задачи, чтобы открыть редактирование.
- Нажмите чекбокс слева от задачи, чтобы завершить или восстановить её.
- Кнопка редактирования появляется при наведении на строку задачи.
- На странице “Неделя” задачи можно перетаскивать в другой день.

## Приоритеты

- Красный — самый высокий приоритет.
- Жёлтый — высокий приоритет.
- Зелёный — средний приоритет.
- Синий — обычный приоритет.

## Категории

- Категорию можно создать из кнопки в верхней части боковой панели или из блока “Категории”.
- Для категории можно выбрать цвет, хэштег или эмодзи.
- Избранные категории отображаются отдельным блоком.
- Счётчики рядом с категориями показывают количество задач.
- В настройках можно выбрать сортировку категорий и управлять отображением счётчиков.

## Заметки

- Внизу каждой страницы есть поле “Заметки”.
- Заметки сохраняются автоматически.
- Заметки разделены по контексту: у “Сегодня”, “Недели”, “Входящих” и каждой категории свой текст.
- В настройках можно изменить лимит строк заметок и интервал автосохранения.

## Поиск

- Нажмите “Поиск” в боковой панели.
- Поиск находит задачи, категории и недавно открытые разделы.
- Нажатие на результат открывает нужную задачу или раздел.

## Вкладки

- Вкладки помогают держать открытыми несколько разделов.
- В настройках можно включить или выключить tab-bar.
- Можно настроить восстановление открытых вкладок после запуска приложения.

## Меню страницы

- Кнопка с тремя точками вверху страницы открывает действия текущего раздела.
- Для обычных разделов доступна очистка задач.
- Для страницы категории доступно удаление категории.
- Для категории также есть кнопка добавления или удаления из избранного.

## Telegram

- Telegram-бот работает только пока Afterlight запущен на компьютере.
- Юзернейм бота: @afterlight_task_bot.
- После подключения напишите боту `/start`.
- Можно отправить обычный текст, и бот создаст задачу.
- Примеры сообщений:
- `Купить молоко`
- `Купить молоко сегодня`
- `Купить молоко завтра 18:00`
- `Сдать отчёт 12.06 14:30`
- `Позвонить клиенту #Работа`
- Через бота можно посмотреть списки задач, выполнить задачу, удалить задачу и создать категорию.

## Настройки

- В разделе “Основное” находятся запуск, вкладки, сортировка задач, настройки страницы “Сегодня”, настройки недели и заметок.
- В разделе “Язык” можно переключить русский и английский интерфейс.
- В разделе “Тема” можно выбрать светлую или тёмную тему.
- В разделе “Боковая панель” можно включить автосворачивание и счётчики.
- В разделе “Уведомления” настраиваются системные напоминания Windows.
- В разделе “Интеграция Telegram” подключается локальный бот.
- В разделе “Резервные копии” находятся экспорт, импорт, открытие данных и автоматические бэкапы.

## Резервные копии

- JSON подходит для переноса задач обратно в Afterlight.
- CSV подходит для просмотра задач в таблицах.
- Автоматические SQLite-копии помогают защититься от случайной потери данных.

## Важное ограничение MVP

Afterlight сейчас работает как локальное приложение. Данные и Telegram-бот завязаны на запущенный Afterlight на вашем компьютере.
