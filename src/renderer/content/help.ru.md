# Руководство пользователя

Afterlight помогает быстро фиксировать задачи, раскладывать их по дням и категориям, вести короткие заметки, искать нужную информацию и управлять частью задач из Telegram.

## Основные разделы

- **Входящие** - место для задач без конкретного дня. Сюда удобно складывать всё, что нужно разобрать позже.
- **Сегодня** - задачи текущего дня приложения. Начало дня можно настроить через время обновления страницы "Сегодня".
- **Неделя** - планирование текущей недели по дням. Задачи недели без даты находятся в распределителе и могут быть перенесены на конкретный день.
- **Категории** - контексты вроде Учёба, Дом, Работа или Проект. Категории можно добавлять в избранное.

## Работа с задачами

- Нажмите "Добавить задачу", чтобы создать задачу в текущем разделе.
- У задачи можно указать название, описание, дату, время дедлайна, приоритет и категорию.
- Нажмите на название задачи, чтобы открыть редактирование.
- Нажмите чекбокс слева от задачи, чтобы завершить или восстановить её.
- Кнопка редактирования появляется при наведении на строку задачи.
- На странице "Неделя" задачи можно переносить между днями.
- Если включена настройка показа просроченных задач, активные просроченные задачи могут отображаться на странице "Сегодня".

## Приоритеты

- Красный - высший приоритет.
- Жёлтый - средний приоритет.
- Зелёный - низкий приоритет.
- Синий - обычная задача без явного приоритета.

## Категории

- Категорию можно создать из sidebar или из блока "Категории".
- Для категории можно выбрать цвет, хэштег или emoji-иконку.
- Избранные категории отображаются отдельным блоком.
- Счётчики рядом с категориями показывают количество задач.
- В настройках можно выбрать сортировку категорий и управлять отображением счётчиков.
- Emoji-иконки категорий отображаются в sidebar, tab-bar, задачах и поиске.

## Заметки

- Внизу каждой страницы есть поле "Заметки".
- Заметки сохраняются автоматически.
- У "Сегодня", "Недели", "Входящих" и каждой категории свой отдельный текст заметок.
- В настройках можно изменить лимит строк заметок и интервал автосохранения.

## Поиск

- Нажмите "Поиск" в боковой панели.
- Поиск находит задачи, категории и недавно открытые разделы.
- Нажатие на результат открывает нужную задачу, категорию или раздел.
- Блоки "Недавно открытые" и "История" помогают быстро вернуться к последним рабочим контекстам.

## Вкладки

- Вкладки помогают держать открытыми несколько разделов.
- В настройках можно включить или выключить tab-bar.
- Можно настроить восстановление открытых вкладок после запуска приложения.
- Вкладки категорий показывают выбранную иконку категории.

## Меню страницы

- Кнопка с тремя точками вверху страницы открывает действия текущего раздела.
- Для обычных разделов доступна очистка задач.
- Для страницы категории доступно удаление категории.
- Для категории также есть кнопка добавления или удаления из избранного.

## Уведомления

- В разделе "Уведомления" настраиваются системные уведомления Windows.
- Можно включить напоминания о дедлайнах.
- Можно настроить, за сколько минут до дедлайна отправлять уведомления Windows и Telegram.
- Можно включить уведомление перед ежедневным обновлением страницы "Сегодня".
- Можно настроить интервал повторных уведомлений о просроченных задачах.

## Telegram

Telegram-интеграция находится в настройках: **Интеграция Telegram**.

Доступны два режима:

- **Свой токен** - локальный бот, созданный пользователем в BotFather. Работает, пока Afterlight запущен на компьютере.
- **Afterlight Bot** - режим подключения к боту `@afterlight_task_bot`. Этот режим используется для работы с Afterlight через общий бот, если подключение настроено и активно.

После сохранения настроек отправьте боту команду `/start <код>`, которую показывает Afterlight.

Бот умеет:

- добавлять задачи обычным сообщением;
- показывать входящие, сегодня, неделю и категории;
- создавать категории;
- добавлять задачи в конкретную категорию;
- выполнять, восстанавливать и удалять задачи;
- выбирать русский или английский язык;
- показывать подсказки по формату задач.

## Формат сообщений Telegram

Можно отправить обычный текст, и бот создаст задачу:

- `Купить молоко`
- `Купить молоко сегодня`
- `Купить молоко завтра 18:00`
- `Сдать отчёт 12.06 14:30`
- `Позвонить клиенту #Работа`
- `Домашка завтра #Учёба`

Бот понимает:

- даты: `сегодня`, `завтра`, `12.06`, `12.06.2026`;
- время: `09:30`, `18:00`;
- категории через хэштег: `#Работа`, если такая категория уже существует;
- приоритет в начале сообщения.

Приоритеты в Telegram:

- `1 Срочно отправить отчёт` - высший приоритет;
- `2 Сделать домашку` - средний приоритет;
- `3 Посмотреть материалы` - низкий приоритет;
- `Купить молоко` - обычная задача без явного приоритета.

Также можно использовать команду `/add текст задачи`.

## Настройки

- В разделе "Аккаунт" можно изменить имя, название рабочего пространства, email и аватар.
- В разделе "Основное" находятся запуск, вкладки, сортировка задач, настройки страниц "Сегодня" и "Неделя", заметки и поведение интерфейса.
- В разделе "Язык" можно переключить русский и английский интерфейс.
- В разделе "Тема" можно выбрать светлую или тёмную тему.
- В разделе "Боковая панель" можно включить автосворачивание и счётчики.
- В разделе "Уведомления" настраиваются системные напоминания.
- В разделе "Интеграция Telegram" подключается Telegram-бот.
- В разделе "Резервные копии" находятся экспорт, импорт, открытие папки данных и автоматические бэкапы.

## Резервные копии

- JSON подходит для переноса задач обратно в Afterlight.
- CSV подходит для просмотра задач в таблицах.
- Автоматические SQLite-копии помогают защититься от случайной потери данных.
- В настройках можно открыть папку данных и файл базы данных.

## Важное ограничение MVP

Afterlight сейчас остаётся локальным приложением. Основные данные хранятся на вашем компьютере в SQLite. Облачная синхронизация между устройствами и полноценный сервер пользовательских аккаунтов пока не входят в MVP.
