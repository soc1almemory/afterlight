# Afterlight v0.2.5

Обновление v0.2.5 переводит режим Afterlight Bot на Cloudflare, улучшает Telegram-синхронизацию и исправляет проблемы с датами, временем и историей изменений.

## Telegram и Afterlight Bot
- Режим **Afterlight Bot** переведён на Cloudflare Worker и теперь работает через общий серверный мост для `@afterlight_task_bot`.
- Адрес Cloudflare-сервера зашит в приложение по умолчанию, ручной ввод адреса сервера убран из настроек.
- Старые локальные адреса Afterlight Bot server больше не используются в режиме Afterlight Bot.
- Добавлена нормальная многопользовательская схема: разные пользователи могут подключать свои рабочие пространства к общему боту через индивидуальные коды привязки.
- Авторизация бота обновлена: сначала отправляется `/start`, затем 6-значный код отдельным сообщением.
- Добавлен сброс Telegram-сессий для очистки подключённых чатов и ожидающих авторизаций.
- Статус Telegram-интеграции теперь показывает количество подключённых чатов в режиме Afterlight Bot.

## Синхронизация через Cloudflare
- Добавлена синхронизация задач и категорий между локальным приложением и Afterlight Bot через Cloudflare Worker.
- Задачи и категории, созданные в Telegram, появляются в приложении без запуска локального bot-server.
- Удалённые задачи синхронизируются между приложением и ботом через список удалений.
- Фоновая синхронизация больше не перезаписывает время изменения задач и категорий.
- Время последнего изменения теперь обновляется только при реальном изменении данных, а не при пустом sync.

## Даты и время
- Afterlight Bot теперь получает системную таймзону приложения и считает `сегодня`, `завтра` и текущую неделю относительно времени пользователя.
- Исправлено отображение текущей недели в Telegram-боте после перехода на Cloudflare.
- В выборе дня недели в Telegram текущий день помечается отдельно.
- Исправлена подсветка текущего дня на странице «Неделя» в приложении.
- Страница «Неделя» теперь ориентируется на календарную неделю Windows, а не на настройку времени обновления страницы «Сегодня».
- Исправлены ложные статусы «Изменено только что», появлявшиеся из-за фоновой Telegram-синхронизации.
- Исправлены некорректные даты и время в поиске, истории и недавно открытых разделах после синхронизации.

## Интерфейс Telegram-интеграции
- Из настроек Telegram убрано поле адреса Afterlight Bot server.
- Тексты статуса адаптированы под Cloudflare-режим вместо локального сервера.
- В режиме Afterlight Bot больше не отображается `chat_id`, он оставлен только для режима «Свой токен».
- Улучшена локализация сообщений статуса Telegram-интеграции.
- Обновлены help-файлы с актуальным описанием Cloudflare-режима.

## Telegram-бот
- Убраны лишние inline-кнопки навигации из сообщений бота: навигация теперь находится в основном Telegram-меню.
- Восстановлена автоматическая очистка старых сообщений бота после действий пользователя.
- Бот больше не очищает чат полностью так, чтобы пустой Telegram-чат закрывался на мобильных устройствах.
- Улучшено распознавание дат и времени в текстовых задачах.
- Добавлена поддержка приоритетов через цифру в начале сообщения: `1`, `2`, `3`.
- Обновлены русские подсказки бота с примерами приоритетов, дат, времени и категорий.
- Добавлены Telegram-напоминания о дедлайнах при включённых уведомлениях.

## Стабильность
- Worker получил мягкую миграцию схемы D1 для новых полей таймзоны.
- Серверная логика стала устойчивее к старым сохранённым настройкам и legacy-адресам.
- Исправлены проблемы, при которых уже подключённый бот мог показывать неверный статус подключения.
