# Afterlight v0.2.4

Обновление v0.2.4 улучшает страницу «Неделя», автоматическое обновление и добавляет анимацию загрузки.

## Обновления приложения
- Добавлена анимация загрузки обновления.
- Уведомление об установке обновление теперь может появляться повторно.

## Раздел «Неделя»
- В разделе «Неделя» распределитель без даты заменён на «Входящие».
