# Afterlight v0.2.2

Версия 0.2.2 улучшает интеграцию с Telegram и настройки задач, а также добавляет автоматические обновления приложения.

## Обновления приложения
- Добавлена автоматическая проверка обновлений для установленной Windows-версии Afterlight.
- Новые версии скачиваются из GitHub Releases репозитория Afterlight.
- Приложение показывает небольшое всплывающее уведомление только после загрузки обновления, когда оно готово к установке.
- Из уведомления можно перезапустить Afterlight, чтобы завершить установку.

## Настройки
- Доработана локализация telegram-status-message.
- В разделе **Основное -> Сортировка задач** добавлена настройка, которая перемещает выполненные задачи в конец списка.

## Help и Changelog

- Обновлено руководство пользователя и список изменений.

## Telegram
- Бот автоматически создаёт указанную категорию при добавлении задачи, если её не существовало.
- Приложение сохраняет служебные данные Telegram-интеграции при обновлении настроек, чтобы повторное подключение чатов не сбрасывалось.
- Исправлено сохранение данных авторизации Telegram-бота при обновлении статуса и настроек приложения.
- Статус режима Afterlight Bot теперь определяется по реальным авторизованным чатам, а не только по legacy `chat_id`.
- В статус Telegram-интеграции добавлен счётчик подключённых чатов для режима Afterlight Bot.
- Добавлена кнопка сброса Telegram-сессий без отключения интеграции и удаления кода привязки.
- `chat_id` теперь отображается только в режиме **Свой токен**.
- Технические ошибки Telegram в строке статуса теперь локализуются на язык интерфейса.

## Тёмная тема и иконки
- `color-swatch.active` теперь белого цвета (`#ffffff`)
- Подключённая Telegram-иконка в светлой теме теперь использует отдельный светлый вариант.

## Исправления
- Исправлен drag-region панели вкладок, который мешал нормальной работе кнопок.
- Строки `telegram-status` больше не наследуются при переключении режимов бота.
