# Afterlight v0.2.0

Текущее состояние Afterlight: рабочий локальный менеджер задач для Windows с задачами, категориями, заметками, настройками, Telegram-интеграцией, уведомлениями и резервными копиями.

## Основной рабочий процесс

- Добавлены разделы "Входящие", "Сегодня", "Неделя" и страницы категорий.
- Реализованы задачи с названием, описанием, датой, временем дедлайна, приоритетом, статусом и привязкой к категории.
- Добавлено быстрое создание задач из текущего раздела.
- Добавлено редактирование, завершение, восстановление и удаление задач.
- Раздел "Сегодня" учитывает настроенное время обновления и может показывать активные просроченные задачи.
- Раздел "Неделя" показывает задачи с датами текущей недели и задачи недели без даты в распределителе.
- Исправлены счётчики недели: задачи с датой вне текущей недели больше не считаются задачами текущей недели.

## Категории и избранное

- Добавлены категории с цветом, хэштегом или эмодзи-иконкой.
- Добавлено избранное для категорий.
- Добавлены счётчики задач в боковой панели.
- Добавлены настройки сортировки категорий и отображения счётчиков.
- Исправлено отображение эмодзи-иконок категорий во вкладках.

## Заметки

- У каждого основного раздела и каждой категории есть отдельное поле заметок.
- Добавлено автосохранение заметок.
- Добавлена настройка лимита строк заметок.
- Добавлено отображение времени последнего изменения страницы.

## Интерфейс

- Добавлена панель вкладок.
- Добавлено восстановление открытых вкладок после запуска.
- Добавлены настройки стартового раздела и последнего состояния окна.
- Добавлена светлая и тёмная тема.
- Добавлена локализация интерфейса на русский и английский язык.
- Добавлено автоматическое сворачивание боковой панели.
- Добавлены плавные анимации popup, dropdown, списков, вкладок, кнопок и рабочих экранов.
- Добавлен полноэкранный режим запуска.
- Убрано устаревшее поле безопасности профиля, потому что приложение пока не имеет настоящей локальной авторизации.

## Поиск

- Добавлен popup поиска по задачам и категориям.
- Добавлена история недавно открытых разделов.
- Поиск умеет быстро переходить к найденной задаче или категории.
- Метаданные раздела "Сегодня" в поиске теперь учитывают настроенное время обновления.

## Настройки Windows

- Добавлены настройки автозапуска вместе с Windows.
- Добавлена настройка запуска в свёрнутом состоянии.
- Добавлена настройка поведения кнопки закрытия: выход, трей или подтверждение.
- Добавлены системные уведомления о дедлайнах, просроченных задачах и обновлении страницы "Сегодня".
- Добавлен системный трей.

## Резервные копии и данные

- Добавлен экспорт задач в JSON.
- Добавлен экспорт задач в CSV.
- Добавлен валидируемый импорт задач из JSON.
- Добавлена кнопка открытия папки данных.
- Добавлена кнопка открытия SQLite-базы.
- Добавлены автоматические резервные копии SQLite.

## Telegram

- Добавлена локальная Telegram-интеграция через токен бота, созданного пользователем в BotFather.
- Бот умеет добавлять задачи обычным текстом из Telegram.
- Бот понимает даты, время и категории через хэштеги.
- Добавлены команды и кнопки Telegram-бота.
- Добавлен выбор языка бота.
- Добавлено создание категорий из Telegram.
- Добавлены списки задач, выполнение и удаление задач прямо из Telegram.
- Добавлена автоочистка старых сообщений бота в чате.
- Telegram-токен хранится через Electron `safeStorage`, если в системе доступно шифрование.

## Технические улучшения

- Данные приложения хранятся локально в SQLite.
- Electron renderer изолирован через preload API.
- Добавлены sandbox для renderer и Content Security Policy.
- Сборка работает через Electron Forge и Vite.
- Проверка TypeScript запускается перед упаковкой.
- Production sourcemap-файлы отключены для релизных сборок.
