# Afterlight v0.2.6

Обновление v0.2.6 улучшает работу с задачами через Telegram, поиск и навигацию, а также исправляет несколько деталей интерфейса.

## Help и Changelog

- Обновлён список изменений.

## Telegram

- Лимит списка задач поднят с 8 до 30.
- Добавлена сортировка задач: активные выше выполненных, затем приоритет **1 → 2 → 3 → без приоритета**, потом дата и время.
- Раздел “Неделя” теперь группируется по дням.
- В выборе дня недели для добавления задачи теперь тоже показываются дни недели, а не только даты.

## Поиск и навигация

- Исправлено поведение: при нажатии на задачу произойдёт переход к источнику.
- Добавлена короткая подсветка задачи при переходе из поиска.
- Добавлена автоматическая прокрутка рабочей области вверх при переходе из поиска.

## Категории и вкладки

- Добавлена автоматическая прокрутка рабочей области вверх при двойном нажатии.

## Дизайн

- Изменены цвета счётчиков в боковой панели.

## Тёмная тема и иконки

- Исправлена иконка задачи в меню поиска при тёмной теме.
