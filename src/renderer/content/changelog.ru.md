# Afterlight v0.2.1

Обновление 0.2.1 закрывает финальные MVP-доработки: улучшает Telegram-интеграцию в приложении, делает интерфейс плавнее, обновляет справку и changelog, исправляет тёмную тему, синхронизацию данных и несколько заметных визуальных багов.

## Интерфейс и анимации

- Добавлены плавные анимации появления popup-окон, dropdown-меню, вкладок, списков, кнопок и переходов между экранами.
- Добавлена красивая анимация перехода из экрана первичной настройки профиля в рабочее пространство и обратно.
- Исправлено мигание аватара при переходе из setup-экрана: выбранный аватар больше не подменяется default-изображением во время анимации.
- Заменена стартовая подпись "Загрузка Afterlight..." на крупный центрированный индикатор загрузки.
- Для тёмной темы у загрузочного экрана используется фон `#242424`.
- Добавлена анимация сворачивания и разворачивания sidebar через `titlebar-sidebar-toggle`.
- Добавлена hover-анимация для логотипа Afterlight в titlebar: базовая иконка плавно заменяется цветной.
- Исправлены размеры и отступы кнопки закрытия настроек, чтобы она совпадала с кнопками закрытия help и changelog popup.
- Исправлен размер текста кнопки "Сбросить" в настройках.

## Настройки

- Версия приложения теперь хранится в отдельном файле и отображается внизу навигации настроек.
- В настройках отображается подпись `Afterlight v0.2.1 © soc1almemory 2026`.
- Добавлены настройки уведомлений:
  - за сколько минут до дедлайна уведомлять через Windows и Telegram;
  - за сколько минут до обновления страницы "Сегодня" уведомлять через Windows;
  - через какой интервал повторять уведомления о просроченных задачах через Windows.
- В разделе "Интеграция Telegram" добавлен переключатель режимов бота.
- Для режима Telegram добавлены компактные кнопки и улучшенная строка статуса.
- Кнопка "Проверить подключение" доступна для обоих режимов Telegram.
- Статус Telegram автоматически обновляется каждые 5 секунд.
- В разделе Telegram отображается username бота `@afterlight_task_bot`.
- Исправлены пустые чекбоксы в тёмной теме: теперь они используют цвет `#1F1F1F`.
- В тёмной теме кнопки удаления профиля и подтверждения удаления стали красными.

## Help и Changelog

- Help popup теперь получает содержимое из внешних `.md` файлов.
- Changelog popup теперь получает содержимое из внешних `.md` файлов.
- Справку и changelog теперь можно обновлять без правки JSX-компонентов.
- Обновлено руководство пользователя: добавлены актуальные разделы про режимы Telegram, уведомления, поиск, резервные копии и формат сообщений боту.

## Telegram

- В настройках добавлен режим "Свой токен" для локального бота пользователя.
- В настройках добавлен режим "Afterlight Bot" для подключения к боту `@afterlight_task_bot`.
- Кнопка Telegram в нижнем меню теперь открывает раздел настроек "Интеграция Telegram" вместо старого popup.
- Telegram-иконка в нижнем меню показывает подключённое состояние через `telegram-icon-connected.svg`.
- Добавлена большая статусная Telegram-иконка в настройках с адаптацией под светлую и тёмную темы.
- Статус подключения теперь учитывает не только `chat_id`, но и реальное состояние подключения.
- Исправлена синхронизация данных между Telegram-интеграцией и открытым окном Afterlight: новые задачи и категории подтягиваются без перезапуска приложения.
- Добавлено наблюдение за локальной базой данных и конфигом Telegram, чтобы интерфейс обновлялся при внешних изменениях.
- Повторная загрузка данных больше не сбрасывает текущую страницу и вкладки.
- Улучшена автоочистка чата Telegram: бот больше не удаляет сообщение, которое прямо сейчас вызвало действие, поэтому чат не закрывается в мобильном Telegram.
- Бот научился определять приоритет задачи из начала сообщения:
  - `1 Текст задачи` - высший приоритет;
  - `2 Текст задачи` - средний приоритет;
  - `3 Текст задачи` - низкий приоритет;
  - без цифры - обычный приоритет.
- Обновлены русские и английские подсказки Telegram-бота по форматированию задач.

## Поиск и навигация

- Даты и время в блоках "Недавно открытые" и "История" теперь обновляются корректно.

- Новые категории и открытые разделы теперь корректно попадают в историю.
