# Afterlight v0.2.8

Обновление v0.2.8 значительно улучшает дизайн всех элементов приложения, исправляет синхронизацию с Telegram-ботом и другие значительные баги.

## Дизайн
- Дизайн приложения значительно обновлён.
- setup-panel в Setup Screen теперь адаптируется по высоте.

## Боковая панель
- Категория, созданная из секции "Избранное", сразу создаётся в избранное.
- Обычная add-кнопка всё ещё создаёт обычную категорию.

## Telegram
- Локальная SQLite теперь хранит удалённые задачи и категории, чтобы server snapshot не мог воскресить их обратно.
- Sync с ботом отправляет deletedTaskIds и deletedCategoryIds.
- Worker хранит удалённые категории, удаляет их на своей стороне и больше не upsert-ит старые версии поверх свежих.
- Merge стал timestamp-aware: более старый snapshot не перетирает более свежие локальные правки.
