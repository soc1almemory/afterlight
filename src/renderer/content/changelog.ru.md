# Afterlight v0.2.1

Обновление 0.2.1 улучшает Telegram-интеграцию в приложении и исправляет синхронизацию данных.

## Настройки

- В разделе "Интеграция Telegram" добавлен переключатель режимов бота.
- Кнопка "Проверить подключение" доступна для обоих режимов Telegram.
- В разделе Telegram отображается username бота `@afterlight_task_bot`.

## Help и Changelog

- Обновлено руководство пользователя.

## Telegram

- В настройках добавлен режим "Свой токен" для локального бота пользователя.
- В настройках добавлен режим "Afterlight Bot" для подключения к боту `@afterlight_task_bot`.
- Статус подключения теперь учитывает не только `chat_id`, но и реальное состояние подключения.
- Исправлена синхронизация данных между Telegram-интеграцией и открытым окном Afterlight: новые задачи и категории подтягиваются без перезапуска приложения.
- Добавлено наблюдение за локальной базой данных и конфигом Telegram, чтобы интерфейс обновлялся при внешних изменениях.
- Повторная загрузка данных больше не сбрасывает текущую страницу и вкладки.
- Улучшена автоочистка чата Telegram: бот больше не удаляет сообщение, которое прямо сейчас вызвало действие, поэтому чат не закрывается в мобильном Telegram.
- Бот научился определять приоритет задачи из начала сообщения:
  - `1 Текст задачи` - высший приоритет;
  - `2 Текст задачи` - средний приоритет;
  - `3 Текст задачи` - низкий приоритет;
  - без цифры - обычный приоритет.
- Обновлены русские и английские подсказки Telegram-бота по форматированию задач.

## Исправления

- Исправлен конфликт визуального состояния Telegram-иконки и текста статуса в настройках.
- Исправлен растягивающийся переключатель режимов Telegram.
- Исправлен фантомный hover первого варианта в переключателе режимов Telegram.
